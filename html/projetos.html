Projetos

Aqui estão meus projetos recentes:

  • Rei Das Motos:
    O Rei das Motos é um sistema de venda de motos desenvolvido para facilitar a divulgação e negociação de motocicletas. A plataforma permite visualizar modelos, preços e informações de forma prática e organizada. Seu objetivo é oferecer uma experiência simples, rápida e moderna para clientes e vendedores.

  • Projeto Social: O Projeto iFrame é uma aplicação desenvolvida para simular a interface de um celular diretamente no navegador. O objetivo do projeto é treinar responsividade e navegação entre páginas utilizando HTML, CSS e iFrames de forma prática e interativa.

  • Projeto Cordel: O Projeto Cordel é uma aplicação desenvolvida para apresentar poemas de forma interativa e visualmente atraente. O objetivo do projeto é promover a cultura literária e oferecer uma experiência única para os usuários.
